# Fashion-House
Fashion House

## Set up server for development 

#### Set up front-end 
run<br>
`cd ./client`<br>
    `npm install`<br>
    `npm start`<br>
http://localhost:5000/

#### Set up back-end
run <br>
`run cd ./server`<br>
    `npm install`<br>
    `npm run dev`
